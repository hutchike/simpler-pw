simpler.pw

How to simply generate a new password:

  1. Choose a public username, e.g. "spock" (it can be any word you like)
  2. Choose a secret password, e.g. "green socks" (keep this secret!)
  3. Visit simpler.pw/pass by entering your public username and secret password
  4. Enter a short memorable word to receive a secure password

Your secret password is not stored anywhere; It is only used to generate a secure password from your memorable word.
